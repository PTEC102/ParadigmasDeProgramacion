# Descargar Netbeans y openJDK en windows

Para descargar el openJDK (que es el que contiene Java) debemos ingresar la siguiete URL (cortesía de Rodolfo Aravena, jefe de laboratorios):

[Descargar openJDK](https://bit.ly/OpenJDKFolder)

Dentro, deben descargar el archivo OpenJDK y seguir las instrucciones que ahi aparecen

```
1. Descomprimir o abrir el archivo zip
2. Copiar la carpeta "java" en C:/
3. Abrir la carpeta "java" y abrir el archivo "install_openjdk.bat"
4. Abrir cmd y ejecutar el comando:
	java -version
5. Si muestra la versión de java quedó instalado correctamente
```

Hecho esto, podemos descargar e instalar netbeans desde su página oficial

[Descargar netbeans](https://netbeans.apache.org/download/index.html)
